# 📊 Marketing Analytics Project – ShopEasy

## 🚀 Overview
This project was developed as part of a **Data Analysis Hackathon by Orange Digital Center**.

The objective was to analyze marketing performance, customer engagement, and customer feedback to generate actionable insights that improve business performance.

---

## 🎯 Business Problem
ShopEasy is facing several challenges:

* 📉 Low conversion rates
* 📉 Decreasing customer engagement
* 💸 Increased marketing spend with low ROI
* 😡 Negative customer feedback trends

---

## 📂 Dataset Description
The dataset includes the following tables:
* Customers
* Products
* Customer Journey
* Customer Reviews
* Engagement Data

---

## 🛠️ Tools & Technologies
* **SQL** → Data cleaning & analysis (using CTEs)
* **Power BI** → Dashboard & visualization
* **DAX** → KPI calculations
* **Excel** → Data preprocessing
